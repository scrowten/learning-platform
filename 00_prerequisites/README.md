# 00 — Prerequisites `[Beginner]`

## Who needs this?

If you're comfortable with matrix operations, basic probability, and Python/numpy/pandas, **skip this section entirely**.

Come back here if you hit a gap while working through a later module.

## Contents

| Subdirectory | Topics |
|---|---|
| `linear_algebra/` | Vectors, dot products, matrix multiplication, eigenvalues, SVD, PCA geometry |
| `probability_statistics/` | Probability rules, distributions, MLE, MAP, Bayesian inference, hypothesis testing |
| `python_numpy_pandas/` | Python essentials, numpy broadcasting, pandas DataFrames, matplotlib basics |

## Skip guide

| Background | Recommendation |
|---|---|
| STEM degree (math/CS/physics) | Skip entirely |
| Know Python but not numpy | Do `python_numpy_pandas/` only |
| Self-taught, shaky on matrix math | Do `linear_algebra/` + `probability_statistics/` |
| Complete beginner | Work through all three in order |

## Prerequisites

None — this is the starting point.
