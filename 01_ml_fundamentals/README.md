# 01 — ML Fundamentals `[Beginner–Intermediate]`

## Overview

Classical machine learning — the foundation that every practitioner needs, and that modern deep learning builds on top of.

## Contents

| Subdirectory | Topics | Difficulty |
|---|---|---|
| `supervised/` | Linear regression, logistic regression, SVMs, decision trees, k-NN, naive Bayes | Beginner |
| `unsupervised/` | k-means, DBSCAN, hierarchical clustering, PCA, ICA, anomaly detection | Intermediate |
| `ensemble_methods/` | Bagging, random forests, AdaBoost, gradient boosting, XGBoost, LightGBM, stacking | Intermediate |

## Skip guide

| Background | Recommendation |
|---|---|
| You've trained ML models for work | Skim `concepts.md`, do the from-scratch notebooks to refresh |
| You use sklearn but don't know the math | Do `concepts.md` carefully, then notebooks |
| New to ML | Full module — don't skip |

## Prerequisites

- Basic Python
- Linear algebra basics (vectors, matrices)
- Probability basics (distributions, MLE)

See `00_prerequisites/` if you need a refresher.
