Concept Q&A

Ask questions about any module — "explain backpropagation like I know linear algebra" or "how does DCF differ from comparable comps?" RAG over your own content via the agentic-ai backend.

Adaptive Learning Paths

User states what they know → AI generates a personalized module sequence. "I know Python and stats but not ML" → custom path skipping prerequisites automatically.

DB needs: user profiles + progress tracking + prerequisite graph
C

Auto-Generated Quizzes

LLM generates questions from concepts.md per module. Tracks what you've answered to surface weak spots. Spaced repetition built on top.

DB needs: quiz items + answer history + spaced-rep schedule

Cross-Domain Concept Linking

"Gradient descent in ML" ↔ "marginal cost optimization in economics" ↔ "momentum in trading." AI surfaces conceptual connections across domains automatically.

DB needs: concept graph with cross-domain edges
E

Progress Tracking + Streaks

Track which modules are done, in-progress, or skipped per user. Dashboard showing coverage across domains. No AI needed — pure data, but enables the features above.

DB needs: user progress table
F

SOTA / Field Update Alerts

AI monitors arxiv / news for each domain's sota.md topics. Flags when a module's content may be outdated. Drafts update suggestions automatically.

DB needs: content version + last-reviewed timestamps
